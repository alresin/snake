# Игра змейка

Классическая игра змейка. Написана на языке python с использованием библиотеки pygame.

## Информация по запуску игры

При запуске файла `main.py` будет предложено ввести размеры игрового поля и скорость змейки. Это является опциональным.

Уравление в игре осуществляется нажатием клавиш WASD.

В игре присутствует пауза, для ее активации/деактивации нажмите клавишу P.
